# EXE2PNG

## Features:
```
* PNG icon generator from EXE files
```
## Dependencies:
```
imagemagick
icoutils
```
## To get started:
* **Download the latest revision**
```
git clone https://github.com/VHSgunzo/exe2png.git
```
* **Switch to tool's directory**
```
cd exe2png
```
* **INSTALL**
```
sudo make install
```
* Usage:
```
┌──[user@arch]─[~]
└──╼ $ exe2png blabla.exe
...
```
